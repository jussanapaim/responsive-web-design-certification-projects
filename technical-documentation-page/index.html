Introdução ao CSS

CSS é uma sigla que significa "Cascading Style Sheets" em inglês, ou seja, "Folhas de Estilo em Cascata" em português. É uma linguagem de marcação utilizada em conjunto com o HTML para controlar a apresentação visual de uma página da web. Em outras palavras, o CSS permite que você defina como os elementos HTML devem ser exibidos, incluindo cores, fontes, margens, tamanhos e posicionamento.

  • Cascata: O princípio da cascata permite que múltiplos estilos se apliquem ao mesmo elemento, com a hierarquia e a especificidade determinando qual prevalece.
  • Herança: As propriedades CSS podem ser herdadas, simplificando a estilização de elementos filhos com base nas configurações dos elementos pais.

Exemplo da sintaxe do CSS: 


          /*Sintaxe Básica*/
          h1 { color: red; font-size: 24px; }
Seletores

Os seletores são ferramentas poderosas que direcionam estilos para elementos específicos. Desde os básicos de tipo, classe e ID até os avançados combinadores, pseudo-classes e pseudo-elementos, proporcionam flexibilidade. Exemplo: 'p' atinge parágrafos, ''.destaque' personaliza elementos destacados, e '#cabecalho' estiliza o cabeçalho.

  • Seletor de Classe (.) e ID (#): Permitem estilizar elementos específicos, proporcionando uma granularidade significativa.
  • Seletor Universal (*): Aplica estilos a todos os elementos, útil para resetar estilos padrão do navegador.

Um exemplo disso na prática: 


            /* Seletor de tipo - Estiliza todos os parágrafos */
            p { color: blue; }

            /* Seletor de classe - Estiliza elementos com a classe 'destaque' */
            .destaque { font-weight: bold; }

            /* Seletor de ID - Estiliza o elemento com o ID 'cabecalho' */
            #cabecalho { background-color: gray; }
Propriedades de Estilo

As propriedades de estilo determinam a aparência dos elementos selecionados. Cada propriedade, como color para cor do texto, 'margin' para margens e 'border' para bordas, personaliza aspectos específicos do design.

  • Propriedade 'color': Essencial para definir a cor do texto em elementos HTML, permitindo uma personalização visual do conteúdo.
  • Propriedade 'margin': Define o espaço entre a borda do elemento e os elementos vizinhos, influenciando o layout geral.

Um exemplo disso na prática: 


            /* Define a cor do texto para verde */
            color: green;

            /* Adiciona uma margem de 10 pixels ao redor do elemento */
            margin: 10px;
    
            /* Define a largura da borda como 2 pixels, sólida e vermelha */
            border: 2px solid red;
Modelo de Caixa

O Modelo de Caixa é essencial para entender o espaço ocupado por elementos. Detalhes como box-sizing afetam como as dimensões são calculadas, enquanto margin, padding e border oferecem controle preciso sobre a disposição e aparência de cada caixa na página.

  • Preenchimento (padding): Define o espaço entre o conteúdo do elemento e sua borda, contribuindo para o espaçamento interno e o alinhamento visualmente agradável.
  • Altura e largura (height e width): Estas propriedades determinam as dimensões do elemento, controlando o tamanho vertical e horizontal, respectivamente.

Um exemplo disso na prática:


            /* Define a largura e altura do elemento, incluindo preenchimento e borda */
            width: 200px;
            height: 100px;
    
            /* Adiciona preenchimento interno ao redor do conteúdo do elemento*/~
            padding: 10px;
    
            /* Adiciona uma borda ao redor do elemento */
            border: 1px solid black;
Layout e Posicionamento

Para criar layouts complexos, o CSS oferece opções avançadas de posicionamento. position: absolute permite colocação exata, enquanto flexbox (display: flex) e grid (display: grid) oferecem sistemas poderosos para organizar e alinhar elementos em duas dimensões, proporcionando designs modernos e responsivos.

  • Flexbox: Introduz um modelo de layout unidimensional, facilitando a criação de designs flexíveis e a distribuição eficiente de espaço entre os itens.
  • Grid: Oferece um sistema bidimensional para alinhamento preciso de elementos, proporcionando maior controle sobre o layout da página.

Um exemplo disso na prática:


          /* Define o posicionamento absoluto do elemento */
          position: absolute;
          top: 50px;
          left: 20px;
    
          /* Define o elemento como uma caixa flexível */
          display: flex;
    
          /* Alinha os itens flexíveis no centro do contêiner */
          justify-content: center;
          align-items: center;
